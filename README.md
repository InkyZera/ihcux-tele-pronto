# Juan Oliveira Meireles

Com botões grandes e mais fáceis de ver assim que você abre o aplicativo, fica perfeito para quem tem visão ruim ou mãos tremulas

Com o primeiro prototipo, é possível ver uma tela inicial diferente. Quando você abre o aplicativo, você tem a opção de pedir ajuda imediata ao invés de procurar no aplicativo todo o que você precisa.
